WHAT


KNXlevelUP system is our know how: package of industrial PLC with software core Codesys in conjunction with visualization server Iridium with frontend and backend on JS code.


HOW


- industrial PLC enables us to create systems with increased reliability, which processes scenario logic with unlimited duration. PLC is original KNX device give you advantage of KNX technology.
- Iridium visualization server in different hardware designs enables us to create functionally and graphically tailor-made interfaces, at the same time providing data firewall between KNX, Modbus TCP and AV/HTTP systems and allows to control any IP devices with multi-language interface.


KEY FUNCTIONS


- Business Rules Engine (BRE) is KNXlevelUP module, which enables to separate all dependencies, procedures and scenarios into a standalone software or hardware module: simplifies and increases control quality of various modes and scenarios. *
*in upcoming release (planed for Q1 of 2019 machine learning is going to be incorporated in this module)
- modular PLC is used as an interface between various engineering systems, which operate on different physical buses: KNX, DALI, Modbus, RS-485 and other.
- logging system with collects, registers, stores and processes alarms and other events
- notifications through email, SMS, SIP call, voice alarm (via multiroom system), Telegram chat via the telegram bot, system logging on user panel
- simplified procedure of system debugging for installers with logging in real time to Telegram-channel OR on user panel with visualization.
- Bring Your Own Device (BYOD) system is essential technology for residential and commercial spaces based on Iridium visualization. It enables use of interfaces with limitations by roles and time on any smartphone or tablets (for example, guest in a hotel room and use of conference room or mansions).
- MS Exchange Server integration - common solution in corporate segment for booking conference rooms from a touch panel or by e-mail. Can be used together with BYOD
- storage of data in either local or cloud database with mathematical processing of data.
- Ad hoc reports from stored data
- A lot of iRidium drivers for variety of device types:
AV systems and receivers, projectors, TV, multiroom systems, SIP driver, fingerprint scanners and lot more AND also native drivers: KNX, Modbus, HDL, SIP, Helvar, Crestron, AMX, BACNet
We keep developing drivers for new devices. We can also develop drivers for any IP devices on demand, if it has published API.
